Very easy to use the plugin for voting.

== Description ==

This plugin is suitable for voting. There is also different colors inside plugin that makes voting more beautiful.
From each computer or more exactly from on IP address you can vote only for once. 
You see results of voting immediatelly after refreshing the page and when you return on previous page, where you voted, a questioning form disapears.



== Installation ==


1. Upload `searching-for-posts.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [useful_poll ID= id] in widget or
   place `<?php echo do_shortcode("[useful_poll ID= id]"); ?>` in index.php or anywhere in your template

== Frequently Asked Questions ==

= Where to start ? = 

Please go to the Add poll subpage. After creating poll votes form shortcode id is displayed automatically in the Settings page.


= On the front end after submit form my answer is not showing immediatelly. Why ? =

You just need to refresh the page and an answer will appear immediatelly.
